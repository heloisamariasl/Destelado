# 🐱 Destelado

Destelado é um jogo 2D desenvolvido em Python por estudantes da cadeira de Introdução à Programação do Centro de Informática da UFPE, onde o jogador controla um gato que se perdeu nas ruas de uma cidade após sua tela de proteção ser deixada aberta.

Agora sozinho em um ambiente urbano hostil, o gato precisa enfrentar desafios, inimigos como cães e diversas surpresas perigosas para tentar encontrar o caminho de volta para casa.

---

## 🎮 Objetivo do jogo

Ajudar o gato a sobreviver e superar obstáculos até conseguir retornar para sua casa.

---

## 🛠️ Tecnologias utilizadas

- Python
- Pygame

---

## 🚀 Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/heloisamariasl/Destelado.git
```

### 2. Entre na pasta do projeto

```bash
cd Destelado
```

### 3. Instale o Pygame

```bash
pip install pygame
```

### 4. Rode o jogo

```bash
python main/main.py
```
